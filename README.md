# Student Performance Analysis Dashboard

## Project Overview
This project explores factors that influence student academic performance using Excel.

The dataset contains over 6,600 student records and includes variables such as study hours, attendance, teacher quality, motivation level, and exam scores.

The goal of the project was to analyze these factors and build an interactive dashboard to uncover insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Dashboard Design

## Key Insights
1. Students who study more hours tend to achieve higher exam scores.
2. Higher teacher quality is associated with improved student performance.
3. Most students fall within the average performance category, suggesting opportunities for academic improvement.
